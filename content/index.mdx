Описание разделов.

1. **О продукте** - содержит общее описание программы, фукциональности версии, назначения и условия применения.

2. **Начало работы с приложением** - в разделе описана установка программы и настройка рабочего места.

3. **Документы** - содержит описание работы с документами.

	3.1 **Подпись и проверка подписи** - описна работа с электронной подписью.

	3.2 **Шифрование и расшифрование** - описана работа по шифрованию и расшифрованию документов.

	3.3 **Архив** - описана работа с архивными документами.

4. **Сертификаты** - раздел содержит информацию по управлению сертификатами и ключевыми контейнерами.

5. **Контакты** - дано описание работы по управлению локальными контактами и контактами адресной книги LDAP.

6. **О программе** - содержит сведения о версии программы, описана работа с журналом событий.
